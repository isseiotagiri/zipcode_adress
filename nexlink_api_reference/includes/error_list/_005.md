## エラーコード: 0050001

> レスポンス

```json
{
  "application_error_code": "0050001",
  "details": [
    {
      "parameter": "file",
      "message": "原稿ファイルを入力してください。"
    }
  ]
}
```



| 項目|内容|
--- | ---
エラーコード|0050001
ステータス|422
発生箇所|FAX原稿登録
発生事由| ・FAX原稿登録APIに渡すパラメータが不正のため<br />・FAX原稿ファイルからウィルスが検知されたため<br />・ファイル名に使用できない文字が含まれてるため<br />・ファイル名が255文字を超えたため
